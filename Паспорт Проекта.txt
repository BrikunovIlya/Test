Паспорт проекта "Metroid-like Game"

1. Название проекта
Metroid-like Game

2. Назначение проекта
Проект представляет собой консольную игру в жанре платформера с элементами экшена, вдохновленную серией Metroid. Игрок управляет персонажем, который должен уничтожить всех врагов на карте, собирая бонусы и избегая смерти.

3. Версия проекта
1.0.1.0.1.0.1

4. Основные характеристики
- Жанр: Платформер/Экшен.
- Платформа: Windows.
- Язык программирования: C++.
- Тип приложения: Консольное приложение.
- Размер карты: 40x20 символов.
- Количество врагов: 3.
- Управление: Клавиатура (A, D, W, S, Q).

5. Структура проекта
5.1. Классы
- Player: Управление игроком (движение, прыжки, здоровье, стрельба).
- Enemy: Управление врагами (движение, здоровье, взаимодействие с игроком).
- Projectile: Управление снарядами (движение, столкновения).
- Game: Основная логика игры (карта, обновление, рендеринг, ввод).

5.2. Основные функции
- Инициализация карты с платформами, стенами и бонусами.
- Обработка ввода пользователя.
- Обновление физики (гравитация, прыжки, коллизии).
- Обновление врагов и снарядов.
- Проверка условий победы/поражения.

6. Используемые технологии
- Язык: C++.
- Библиотеки: iostream, vector, conio.h, windows.h, cstdlib, ctime, algorithm.
- Инструменты: Любой C++ компилятор (например, g++).

7. Ограничения
- Платформозависимость: Работает только на Windows.
- Графика: Используются ASCII-символы в консоли.
- Отсутствие сохранения прогресса.

8. Условия эксплуатации
- Операционная система: Windows.
- Минимальные требования: Процессор 1 ГГц, 512 МБ ОЗУ, консоль с поддержкой ANSI-символов.
- Установка: Компиляция исходного кода в исполняемый файл.

9. Цели проекта
- Создание простой консольной игры для демонстрации механик платформера.
- Реализация базовых игровых механик: движение, прыжки, стрельба, коллизии.
- Обеспечение стабильной работы без утечек памяти.
